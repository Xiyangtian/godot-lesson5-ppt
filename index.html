🚀 今天的新挑战

我们要做一个新武器：Missile（导弹）

Missile 和 Bullet 有什么不同？

  • Bullet：直线飞行，碰到敌人消失
  • Missile：发射后出现，过一会自动爆炸消失
核心问题：Missile 怎么知道自己该消失了？
7

🤔 做 Missile 需要用到什么？

1. Missile 是一个场景
→ 需要 .tscn 文件
我们学过：Player.tscn, Enemy.tscn, Bullet.tscn
2. Missile 要显示图片
→ 需要 Sprite2D 节点
每个场景都有 Sprite2D
3. Missile 要检测碰撞
→ 需要 Area2D 节点
Bullet 和 Enemy 都用 Area2D 检测碰撞
4. Missile 要定时消失
→ 需要 Timer 节点
Enemy 用 Timer 定时生成
5. Missile 要触发消失
→ 需要 Signal
timeout 信号 → queue_free()
6. Missile 要从玩家位置发射
→ 需要 Input + 实例化
按 Z → instantiate → add_child

🎯 Missile 完整流程

📥 触发 explode() 的两种方式

按下 Z ↓ 1. 实例化 Missile missile_scene.instantiate() get_parent().add_child(missile) ↓ 2. Missile 飞行 _physics_process() global_position += direction * speed * delta ↓ ├─→ 3A. 1秒超时 │ Timer 信号触发 │ _on_timer_timeout() → explode() │ └─→ 3B. 碰到 enemy body_entered 信号触发 _on_body_entered(body) → explode()

📤 explode() 内部做了什么

explode() 被调用 ↓ 1. 实例化 Explosion explosion_scene.instantiate() ↓ 2. 设置爆炸位置 explosion.global_position = Missile.global_position ↓ 3. 添加到场景 get_parent().add_child(explosion) ↓ 4. 删除导弹自身 Missile.queue_free()
9

💥 Explosion 爆炸流程

📥 触发 body_entered 的时机

1. Explosion 被实例化 explosion_scene.instantiate() ↓ 2. 添加到场景 get_parent().add_child(explosion) ↓ 3. Enemy 进入爆炸范围 Area2D 检测到碰撞 ↓ 4. body_entered 信号触发 _on_body_entered(body) ↓ 5. 判断是否在 enemy 组 body.is_in_group("enemy") ↓ 6. 销毁敌人 body.queue_free()

📤 Explosion 自动清理

1. Explosion 出现 实例化并添加到场景 ↓ 2. Timer 开始倒计时 Wait Time = 0.2 ↓ 3. 0.2秒后 Timer.timeout 信号触发 ↓ 4. 调用超时回调 _on_timer_timeout() ↓ 5. 删除自身 queue_free()
注意：检测不到敌人时，先检查 Collision Layer / Mask 是否对应。

方案 A：Missile 飞行（模块 9.1）

适用对象：所有学生（推荐先做这个）

  • 创建 Missile.tscn（Area2D + Sprite2D + CollisionShape2D + Timer）
  • Timer：Wait Time = 1，One Shot + Autostart 开启
  • 先连接 Timer 的 timeout 信号，再补充脚本
  • 用 get_parent().add_child(missile) 加到 World 层
extends Area2D

var direction = Vector2.RIGHT
var speed = 400

func _physics_process(delta):
    global_position += direction * speed * delta

func _on_timer_timeout():
    queue_free()
12

方案 B：Explosion 爆炸（模块 9.2）

适用对象：进度快的学生

Explosion.tscn 节点结构：

  • Area2D（根节点）+ Sprite2D
  • CollisionShape2D（圆形，Radius 控制范围）
  • Timer：Wait Time = 0.2，自动删除
  • 连接 Area2D 的 body_entered 信号销毁敌人
extends Area2D

@export var explosion_scene: PackedScene

func explode():
    var explosion = explosion_scene.instantiate()
    explosion.global_position = global_position
    get_parent().add_child(explosion)
    queue_free()

func _on_timer_timeout():
    explode()
13

Player.gd 添加代码

在 Player.gd 中添加 missile 发射逻辑：

  • 用 get_parent().add_child(missile) 加到 World 层
  • 不要加到 Player 下，否则导弹会跟着玩家移动
  • 记得在 Inspector 拖入 Missile.tscn
@export var missile_scene: PackedScene

func _physics_process(delta):
    var direction = Input.get_vector(
        "ui_left", "ui_right",
        "ui_up", "ui_down")

func _input(event):
    if event.is_action_pressed("fire_missile"):
        fire_missile()

func fire_missile():
    var missile = missile_scene.instantiate()
    missile.global_position = global_position
    get_parent().add_child(missile)

模块 9.3：碰到敌人提前爆炸

两种爆炸触发方式：

  • 情况1：1秒后自动爆炸（Timer timeout）
  • 情况2：碰到敌人立即爆炸（body_entered）
给敌人添加 enemy 组，并检查 Layer/Mask 
extends Area2D

@export var explosion_scene: PackedScene

func _on_body_entered(body):
    if body.is_in_group("enemy"):
        explode()

func _on_timer_timeout():
    explode()

func explode():
    var explosion = explosion_scene.instantiate()
    explosion.global_position = global_position
    get_parent().add_child(explosion)
    queue_free()

⚠️ 常见报错

按键没反应
→ missile_scene 没拖入 Inspector
→ 检查 @export 变量是否配置
Missile 不消失
→ queue_free() 没写或位置不对
→ 确认在 signal callback 里
爆炸没效果 / 检测不到敌人
→ CollisionLayer/Mask 没配置
→ Missile 的 Mask 要勾选敌人所在的层
→ 敌人的 Layer 也要正确设置
→ 两者对应上，body_entered 才能触发
Missile 一直存在
→ Timer 没连上 signal
→ 检查 Timer signal 连接
